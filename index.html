We haven’t known each other for long but I already love you. When you push me away you always manage to pull me back. You help me branch out and explore new things. You are a master at everything you try. I am committed to you forever and ever, and I want to merge my life with yours. I always say your name before asking anything of you.


I can read you like a book. I start you up by typing git init. Once I’ve made changes to my file I type git add, then git commit -m followed by a short description to commit the changes I’ve made. If I want to send the changes to a repository, I use the push command. To make a branch, I type git branch and the name of the new branch. To switch branches, I type git checkout and the name of the branch I want to switch to. I can merge two branches with the simple command of git merge and the name of the branch to be merged. I can delete a branch with the command git -D.


I love you with all my heart, and I need you for everything I do in my love.


Forever & always,


Ashley
